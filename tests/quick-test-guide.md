## 🚀 快速測試清單

### ✅ 第 1 步：基礎功能（1分鐘）

開啟：`http://localhost:8000`

- [ ] 頁面載入成功
- [ ] 看到 Logo 和標題
- [ ] 看到 6 個導覽標籤
- [ ] 看到 5 個主題卡片
- [ ] 檢查瀏覽器分頁顯示 favicon（燈籠圖示）

**✅ 通過 / ❌ 失敗**

---

### ✅ 第 2 步：頁面導覽（1分鐘）

依序點擊所有導覽標籤：

1. [ ] **首頁** → 顯示主題網格
2. [ ] **傳統乞龜** → 顯示2個按鈕
3. [ ] **擲筊杯** → 顯示活動介紹
4. [ ] **廟會模型** → 顯示展覽介紹
5. [ ] **龍興市集** → 顯示市集介紹
6. [ ] **觀音燈會** → 顯示燈會介紹

**✅ 通過 / ❌ 失敗**

---

### ✅ 第 3 步：子頁面導覽（1分鐘）

1. [ ] 點擊「傳統乞龜」→「閱讀介紹」
   - 應顯示 5 個內容區塊
   - 底部有行動呼籲區域

2. [ ] 點擊「立即報名參加」或「前往報名」
   - 應顯示報名表單
   - 包含姓名和電話欄位

**✅ 通過 / ❌ 失敗**

---

### ✅ 第 4 步：表單驗證（1分鐘）

在報名頁面：

1. [ ] 姓名輸入「王」，點擊電話欄位
   - 應顯示錯誤訊息

2. [ ] 姓名輸入「@#$」，點擊電話欄位
   - 應顯示錯誤訊息

3. [ ] 姓名輸入「王小明」，點擊電話欄位
   - 錯誤訊息應消失

4. [ ] 電話輸入「123」，點擊姓名欄位
   - 應顯示錯誤訊息

5. [ ] 電話輸入「0912345678」，點擊姓名欄位
   - 錯誤訊息應消失

**✅ 通過 / ❌ 失敗**

---

### ✅ 第 5 步：404 錯誤處理（30秒）

1. [ ] 在網址列輸入：`http://localhost:8000#invalid-page`
   - 應顯示 404 錯誤頁面
   - 顯示大型「404」數字
   - 有「返回首頁」按鈕

2. [ ] 點擊「返回首頁」
   - 應跳轉到首頁

**✅ 通過 / ❌ 失敗**

---

### ✅ 第 6 步：管理者頁面（30秒）

1. [ ] 前往：`http://localhost:8000#admin`
   - 應顯示「請先登入」
   - 自動跳轉到登入頁面

2. [ ] 前往：`http://localhost:8000#admin-login`
   - 應顯示登入表單

**✅ 通過 / ❌ 失敗**

---

### ✅ 第 7 步：響應式設計（30秒）

開啟開發者工具（F12）→ 裝置模式

1. [ ] 選擇「iPhone SE」
   - 所有內容正常顯示
   - 無橫向滾動

2. [ ] 選擇「iPad」
   - 排版適當

3. [ ] 切換回桌面
   - 顯示完整版面

**✅ 通過 / ❌ 失敗**

---

### ✅ 第 8 步：動畫效果（30秒）

1. [ ] 在首頁點擊不同標籤
   - 頁面有淡入效果
   - 卡片有滑入效果

2. [ ] 滑鼠移到按鈕和卡片上
   - 有 hover 效果
   - 過渡平順

**✅ 通過 / ❌ 失敗**
